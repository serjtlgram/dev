# Serj.Dev — Vibe Coder Portfolio

> **Premium digital portfolio** showcasing real-world vibe-coded products built with AI-assisted engineering.

## 🚀 Live Features

- **Multi-language support** — EN, ES, PT, UK, RU
- **4 theme presets** — Cyber Dark, Clean Light, Neon Synth, Amber Deus
- **Interactive terminal** — type `help`, `projects`, `skills`, `sudo hack`
- **Canvas constellation** — GPU-accelerated particle system with mouse interaction
- **3D tilt project cards** — hologram effects on hover
- **Orbital tech galaxy** — animated tech stack visualization
- **Smooth scroll** — lerp-based scroll interpolation

## 🛠 Tech Stack

- **React 19** + **TypeScript**
- **Vite 8**
- **Tailwind CSS v4**
- **Lucide React** icons
- **Google Fonts** — Inter + JetBrains Mono

## ⚡ Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173)

## 📦 Build for Production

```bash
npm run build
npm run preview
```

## 📂 Project Structure

```
src/
  App.tsx       # Main portfolio component (all-in-one)
  index.css     # Global styles + Tailwind v4
  main.tsx      # React entry point
index.html      # HTML shell with SEO meta tags
```
